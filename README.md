# go-mock-json
A cross-platform minimal JSON mock server written in golang

## Usage
```
Usage of ./go-mock-json:
  -endpoint string 
        Endpoint to serve json at, eg: /api/json  (default "/")

  -json string
    	Path to json file to serve
        
  -port int
    	Port to bind mock json server (default 8080)
        
  -show
    	Pretty Print JSON
```
